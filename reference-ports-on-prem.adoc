= 本機控制台代理的連接埠
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
當在本機 Linux 主機上手動安裝時，控制台代理使用_入站_連接埠。請參考這些連接埠以進行規劃。

這些入站規則適用於所有NetApp Console部署模式。

[cols="10,10,80"]
|===
| 協定 | 港口 | 目的 


| HTTP | 80  a| 
* 提供從客戶端 Web 瀏覽器到本機使用者介面的 HTTP 訪問
* 在Cloud Volumes ONTAP升級過程中使用




| HTTPS | 443 | 提供從客戶端 Web 瀏覽器到本機使用者介面的 HTTPS 訪問 
|===
