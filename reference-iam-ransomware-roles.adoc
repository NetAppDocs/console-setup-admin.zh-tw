= NetApp Console的勒索軟體恢復存取角色
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
勒索軟體復原角色為使用者提供對NetApp Ransomware Resilience的存取權限。勒索軟體復原能力支援以下角色：

* 勒索軟體抵禦能力管理員
* 勒索軟體復原力檢視器
* 勒索軟體復原力用戶行為管理員
* 勒索軟體恢復用戶行為檢視器


角色賦予對設定和操作的不同存取權限。若要設定或查看可疑的使用者行為活動，您必須使用其中一個使用者行為角色。
