= NetApp控制台中的NetApp災難復原角色
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
您可以為使用者指派以下角色，以便他們存取控制台內的NetApp災難復原。災難復原角色可讓您靈活地為使用者指派特定於他們需要在組織內完成的任務的角色。如何分配角色取決於您自己的業務和儲存管理實踐。

災難復原使用以下角色：

* *災難復原管理員*：執行任何動作。
* *災難復原故障轉移管理*：執行故障轉移和遷移。
* *災難復原應用程式管理員*：建立複製計劃。修改複製計劃。開始測試故障轉移。
* *災難復原檢視器*：僅查看資訊。
